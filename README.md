# Front-end Boilerplate of the Exabook social network

### Installing
 1. npm i
 2. npm start

#### log in and default page: 
```
http://localhost:3000/
```
#### dashboard: 
```
http://localhost:3000/dashboard/
```
#### events: 
```
http://localhost:3000/events/
```
#### marketplace: 
```
http://localhost:3000/marketplace/
```
